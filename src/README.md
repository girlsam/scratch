# A React application built from scratch

This React application was built without using `create-react-app`. The repo's name indicates the intention of the application without having a sense of its direction.

## Quick Start

### System Requirements
* node 8+
* npm 6+

### How to run the application
1. Clone the repo and install the dependencies
```
npm install
```
1. Once the dependencies have been installed, you can run the application locally. The application is served using Webpack's `devServer`, and can be started with this command:
```
npm start
```
